# F1 2026 Race Calendar

A fast, lightweight web app showing the complete 2026 Formula 1 race calendar with all 24 rounds.

**Live site:** https://globalroo.github.io/helloworld/

## Features

- **Session times** — Expand any race to see FP1, FP2, FP3, Qualifying, Sprint, and Race times in your local timezone (plus track-local time)
- **Countdown timer** — Live countdown to the next race
- **Season progress** — Visual progress bar tracking how far through the season we are
- **Calendar export** — Download .ics files to add race weekends to your calendar
- **Search & filter** — Find races by name/location, or filter by upcoming and sprint weekends
- **Dark/light theme** — Toggle between themes, preference is saved
- **PWA support** — Installable as a Progressive Web App with offline caching via service worker
- **Accessible** — Keyboard navigable, screen reader friendly, respects reduced motion preferences
- **Zero dependencies** — Single HTML file with inline CSS/JS, loads race data from a JSON file

## Tech Stack

Vanilla HTML, CSS, and JavaScript. No build step, no frameworks, no dependencies.
