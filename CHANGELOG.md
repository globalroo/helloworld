# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-04

### Added

- **2026 Race Calendar**: Complete schedule of 24 races with interactive, accessible race cards.
- **Next Race Hero**: Featured section with a live countdown timer to the next Grand Prix.
- **Dual Timezone Display**: Session times shown in both your local timezone and each track's local timezone.
- **Add to Calendar**: Per-race ICS export to sync sessions with personal calendars.
- **Circuit Information**: Track name, length, laps, DRS zones, circuit type, and debut year for every race.
- **Country Flags**: Visual flag indicators for each event.
- **Search & Filter**: Search by race or location with Upcoming and Sprint filters; URL state persistence for deep-linking.
- **Theme Toggle**: Dark and light mode with automatic preference persistence.
- **Season Progress**: Visual progress bar tracking the current round and overall season completion.
- **PWA Support**: Service worker for offline access and web app manifest for home screen installation.
- **Accessibility**: Semantic HTML, ARIA states, keyboard navigation, skip links, WCAG AA contrast, and reduced motion support.
- **F1.com Links**: Quick links to official Formula 1 race pages.
- **Smooth Animations**: Staggered card entrance and CSS grid expand/collapse transitions.
